# Attenuator playground

![Otoroshi Biscuit Attenuator Inspector playground](/img/attenuator-playground.svg)]

The **Attenuator Playground** allows users to experiment with **Attenuation** in the context of Eclipse Biscuit Tokens. 

But before diving into our tutorial, what is attenuation process of a token ??

Attenuation is the process of reducing or limiting authority in a decentralized system, and the Attenuator Playground provides a visual and interactive environment to test the effects of attenuation on tokens.

Our tool allows you to visually explore the list of 'checks' ( = restrictions) and generate an attenuated token based on the previous one, incorporating the conditions of these checks.
