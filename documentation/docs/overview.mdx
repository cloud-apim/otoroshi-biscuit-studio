# Overview

Otoroshi Biscuit Studio is an Otoroshi extension that allows you to use and integrate Biscuit Tokens into your Gateway.

## Supported Entities

* Biscuit KeyPairs
* Biscuit Verifiers

## Supported Plugins

* Verifiers
* Attenuators
